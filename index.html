Welcome to TheraSpeak

TheraSpeak is a cutting-edge platform designed to transcribe and generate prompts from live audio transcriptions of suicide hotline calls. Our mission is to support people with mental health issues by providing hotline workers with empathetic responses.

Our Solution

🔊

We generate live audio transcriptions of the hotline calls using Amazon Web Services (AWS) and Assembly AI.

📞

We use Twillo to effectively process the call management backend.

🤖

We used ChatGPT OpenAI API to generate prompts based on the live transcription text.

📱

We used Metaphor's API to generate resources the user can use and send them the resources via Twillo's texting service.

👍

We trained a reinforcement learning model to improve prompt generation using 👍 and 👎 buttons for user feedback.
